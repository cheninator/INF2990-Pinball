void FacadeModele::tournerSelectionSouris ( int  x1,
int  y1,
int  x2,
int  y2 
)

Rotation des noeuds selectionnes.

Remarques
Fait une rotation des objets selectionnes autour de leur centre de masse.
L'angle est calcule en fonction du deplacement de (x1,y1) a (x2,y2):
Presentement, l'angle est proportionnel a (y1 - y2).
Pour tester si la rotation est faisable, on prend tous les points des botes
englobantes. Si un seul de ces points, une fois transforme, n'est pas dans la
table, on ne fait pas la rotation.
Paramètres
[in]x1: abcisse du point initial
[in]y1: ordonnee du point initial
[in]x2: abcisse du point initial
[in]y2: ordonnee du point initial
Renvoie
Aucune.
Remarques
: On doit donner des x,y qui ont ete transformes par panel_GL.PointToClient(...)
